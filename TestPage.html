Here is a brief description on how to use CSS "class" & "id"

A class is added to any HTML element (such as h1, h2, p, div, section, ul, li, span, etc...). This gives you the ability to add a label to the element - or a "box" for our analogy. Then in the css file you can say "Any boxes with this label should look like this...".
KEY NOTE: You can have as many elements with the same class as you would like, but you can have only ONE element with the same id. For example you can label as many div elements with the class "indent", but I you can only have one div tag with the id "navigation".

Here is an example of how to add a class to a div element:
<div class="indented">some stuff here</div>

In the CSS file here is how you can modify any element with that class:
.indent{
margin:10px;
background-color:red;
}

Here is an example of how to add an id to a div element:
<div id="navigation">my navigation menu here</div>

In the CSS file here is how you can modify any element with that class:
#indent{
margin:10px;
background-color:red;
}

Notice the main difference? Classes must have "." in the css while id's have "#"
